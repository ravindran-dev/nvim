Opinionated, modular Neovim configuration using lazy.nvim and Catppuccin theme.


## Quickstart
1. Ensure Neovim >= 0.9 installed
2. Place the contents of this repo into `~/.config/nvim/`
3. Start Neovim and run `:Lazy sync`
4. Optional: `:Mason` to manage language servers


## Included features
- Treesitter
- LSP + Mason
- nvim-cmp + Luasnip
- Telescope
- Nvim-tree
- Bufferline + Lualine
- Catppuccin theme
